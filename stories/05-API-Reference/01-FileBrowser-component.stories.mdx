# `FileBrowser` component

## `FileBrowser` ref methods

This section describes the methods available on the imperative handle (a.k.a. ref) of
the `FileBrowser` component. In your React code, you can acquire handle as follows:

```tsx
import { FileBrowser, FileBrowserHandle, FileList } from 'chonky';

const MyComponent = () => {
    const fileBrowserRef = React.useRef<FileBrowserHandle>(null);

    return (
        <FileBrowser ref={fileBrowserRef} files={[]}>
            <FileList />
        </FileBrowser>
    );
};
```
